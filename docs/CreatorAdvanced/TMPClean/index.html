TMPClean

This will clean your TMP folder and get everything reset from what your file manager did. This is best to do right before you upload your avatar.

Using The Script​

After choosing what type of clean you want to do it will automatically clean everything up!

Hard Clean will remove all your file size tags and clean your TMP folder.

Soft Clean will just clean your TMP folder.
